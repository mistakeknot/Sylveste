# fd-iznik-underglaze-firing — Findings

Lens: Iznik underglaze firing sequence, Kara Memi workshop c. 1555-85. Mechanism: pigments differ in thermal tolerance. Cobalt survives any kiln; Armenian-bole red collapses above ~900C and must enter last or be withheld. Once the saggar is closed the sequence is irreversible — wrong order fuses a year of labor into slag.

## Verdict, one line

Sylveste currently plans to fire its entire catalog in a single public kiln. Most of the distinctive pigments are Armenian-bole-fragile without a bisque layer; chromium bleed from the 64-plugin inventory will contaminate every sharp claim adjacent to it. Cancel the plan to fire the whole piece. Fire a three-pigment panel.

## Findings

### P0 — Chromium bleed from `64-plugin Interverse enumeration` adjacent to sharp primitives
**Location:** target-brief §"What Sylveste Is" lines 54-55; README.md architecture table; `docs/sylveste-vision.md` capability mesh.

Any technically-serious reader encountering the 64-plugin inventory on the first screen will read EVERY claim on that screen as platform-slideware, including the genuinely durable ones. In kiln terms: the chromium in the plugin count bleeds across the saggar and discolors the cobalt claims adjacent to it. You lose OODARC, "wired or it doesn't exist," and the closed-loop receipt in the same firing — not because they are weak, but because they share thermal space with an unsupported breadth claim.

**Action this week:** remove the Interverse enumeration from README first screen. Move it to `docs/interverse-index.md` with no cross-link from the top of README. The architecture table itself should list no more than three names: intercore, Clavain, Interspect.

### P0 — Armenian-bole-fragile claim fired without bisque: "Disagreement between models is the highest-value signal"
**Location:** PHILOSOPHY.md distinctive claim #8 (per target-brief line 84).

This claim needs a bisque-layer artifact — a specific published comparison where two models disagreed and the disagreement drove calibration. Without it, the claim reads as consensus-contrarian branding. Armenian bole in a raw kiln becomes black streak.

**Action:** either pair with one artifact (one `/interpeer` transcript showing Claude↔Codex disagreement and what got written to calibration), or pull the claim from the public surface and keep it internal until the artifact exists.

### P0 — Armenian-bole-fragile: "Every action produces evidence. Evidence earns authority."
**Location:** PHILOSOPHY.md distinctive claim #3.

Only Interspect is M2+. The flywheel claim presupposes operational evidence systems; four of the five (Ockham, Interweave, Interop, FluxBench) are M0-M1. Firing this claim publicly with those support systems at arato stage is the textbook chromium-bleed failure: the claim pulls the incompleteness of the support systems into visible light.

**Action:** rewrite the claim as a narrower cobalt statement anchored only to Interspect — e.g. "Routing decisions are calibrated from evidence, not policy. See Interspect." Cut the recursive-flywheel framing from the public surface until two more systems reach M2.

### P1 — Saggar integrity: README wrapping does not protect fragile claims
**Location:** README tagline (target-brief line 31).

The tagline — "where the review phases matter more than the building phases, and the point is not to remove humans from the loop but to make every moment in the loop count" — is a pada-level prose structure. It does not wrap the fragile claims inside a protective register. A cold reader hits that sentence then immediately encounters the 64-plugin saggar with no intermediate shielding.

**Action:** the tagline should name the one cobalt-stable primitive explicitly. Candidate: "Sylveste is an agent platform where the Compound step of OODARC is wired — every run's lesson is written back as calibration that changes the next run."

### P1 — Do not enter this kiln: Garden Salon, Meadowsyn, v1.0 roadmap
**Location:** target-brief §"Two brands, one architecture" lines 66-69; `docs/roadmap-v1.md`.

Garden Salon (not launched) and Meadowsyn (domain-only) are glaze pigments with no body under them. The two-brand framing is decorative. Firing it now, publicly, imposes brand-cognitive load on a reader who has not yet absorbed the first claim. Classic over-ornamentation of a piece that hasn't yet proven it holds water.

**Action:** hide Meadowsyn domain references from README / MISSION / PHILOSOPHY. Move Garden Salon to a post-1.0 notes file with no public link. Keep the monorepo's internal doc; remove external visibility.

### P2 — Under-fired: "Review phases matter more than building phases"
**Location:** README tagline + PHILOSOPHY.md claim #2.

This IS structurally sound — there is actual Clavain machinery that enforces it (brainstorm → strategy → plan → execute → review → ship phase gates). But the claim lives only in tagline prose. It has not been vitrified into a memorable artifact. A reader does not get to feel the phase-gate mechanism; they only read about it.

**Action:** record one 90-second screencast of Clavain refusing to advance from execute → ship because the review phase had no evidence. That's the hamon of this claim. Anchor the tagline to that video.

## Firing Curve (Recommended Kiln Schedule)

A single-firing schedule that bonds without bleeding:

1. **Bisque (structural):** publish one preprint-style methods note on the `estimate-costs.sh` Closed-Loop: cost baseline $2.93, 785 sessions, reproducible command. Plain prose, one table. This is the unglazed but shaped body.

2. **Cobalt line (durable pigment):** README first-screen rewrite names exactly one primitive — either OODARC Compound or "wired or it doesn't exist." Pair it with a link to the bisque artifact. Cobalt survives all firing temperatures; this claim survives skeptical reading because the preprint stands under it.

3. **Last pigment (Armenian-bole, only if ready):** if a second M2 system ships by firing day, apply one additional claim. If not, the single-pigment firing is stronger than a multi-pigment failure.

4. **Hadori (deliberate dulling):** remove visibility of Interverse count, Garden Salon, Meadowsyn, Factory Substrate, FluxBench from all public surfaces until their own firings are prepared.
